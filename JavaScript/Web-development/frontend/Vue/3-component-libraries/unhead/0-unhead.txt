

LIBRARY: UNHEAD-VUE


    Unhead-Vue is a Vue 3 integration library for Unhead, which is a framework-agnostic library for managing the <head> of your HTML document. So Unhead-Vue allows you to dynamically and programmatically manage your document's <head> section (metadata, SEO tags) in Vue 3 apps.


    >> Key Features:

        - Update document title dynamically;
        - Manage meta tags (description, keywords, Open Graph, Twitter Cards, etc);
        - Control link tags (canonical, stylesheets, preload, prefetch);
        - Handle script tags and JSON-LD structured data;
        - Manage viewport and other head elements.


    >> For SEO Optimization:

        - Built-in support for common SEO patterns;
        - Structured data (JSON-LD) helpers;
        - Social media meta tags;
        - Language and charset management.


    >> Installation:
        
        $ npm install @unhead/vue

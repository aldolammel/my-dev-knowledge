

AXIOS: INTEGRATION WITH VUE.JS:

    >> Axios installation:
        /JavaScript/NodeJS/component-libraries/axios-api/axios.txt

    >> Integration with Vue:

        >> For SPA projects:

            1) In front-end .env file (DON'T in back-end .env), add these lines below:
                VITE_PROD_BASE_URL=https://client-main-domain.com
                VITE_BASE_URL=http://localhost:<django's port>  # Under Production, replace it by VITE_PROD_BASE_URL value!

                >> CRITICAL: Vue with Vite mandatorily require env vars with 'VITE_' prefix!

                >> Need to create a front-end .env file?
                    /Environment-Variables/env-for-local/in-frontend/.env

            2) In the front-end main-app-js file, include:
                
                // Importing for SPA:
                ...
                import axios from 'axios';

                // Declarations:
                ...

                // Other configs:
                // Vite automatically searches for VITE_ prefix env vars:
                axios.defaults.baseURL = import.meta.env.VITE_BASE_URL;  // Where Django's running currently!

                // Installing Vue plugins:
                ...

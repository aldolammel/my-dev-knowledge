

SWITCHING THE NODE.JS VERSION:


    >> Using NVM (Node Version Manager):
        
        >> To a specific Node.js version:
            # to install and use the v24.x:
            $ nvm install 24
            $ nvm use 24
            $ node -v


    >> Using N (another and simpler Node Version Manager):

        >> To a specific Node.js version:
            # to install and use the v24.x:
            $ sudo n xxxxxxxxxxxxxxxxxxxxxxxxxx
            $ sudo n xxxxxxxxxxxxxxxxxxxxxxxxxx
            $ node -v

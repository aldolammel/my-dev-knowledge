

PRETTIER FORMATTER: INTEGRATION

    >> Using VSCode:

        1) Install the Prettier extension for the IDE!

        2) Make sure your VSCode user profile is working properly with JS:

            // CUSTOM CODING BEHAVIORS > JAVASCRIPT:
            "[javascript]": {
                "editor.defaultFormatter": "esbenp.prettier-vscode",
                "editor.tabSize": 2,
                "editor.indentSize": "tabSize",
                "editor.insertSpaces": true,
                "editor.formatOnSave": true,
            },

        3) Add the next lines in the /project/.vscode/settings.json:

            ATTENTION: if you're integrating ESLint to the Prettier, and the ESLint is already
                        as the JS default formatter, just skip the JS settings here, leaving it
                        to ESLint!


            // JAVASCRIPT SETTINGS
            "[javascript]": {
                "editor.defaultFormatter": "esbenp.prettier-vscode",
                ...
            },

            // CSS SETTINGS
            "[css]": {
                "editor.defaultFormatter": "esbenp.prettier-vscode",
                ...
            },

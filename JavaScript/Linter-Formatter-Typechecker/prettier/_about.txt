    - JavaScript (including experimental features)
    - JSX
    - Angular
    - Vue
    - Flow
    - TypeScript
    - CSS, Less, and SCSS
    - HTML
    - Ember/Handlebars
    - JSON
    - GraphQL
    - Markdown, including GFM and MDX v1
    - YAML

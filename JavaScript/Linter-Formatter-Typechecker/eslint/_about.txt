

LINTER: ESLINT

    ESLint is a static code analysis tool for JS. Its primary purpose is to find and fix problems in your code. It doesn't run your code; instead, it scans your source files and flags
    issues based on a set of predefined rules.

    Checks: finds errors and enforces style rules.

    >> It's a linter natively to:
        - JavaScript
        - JSON
    
    >> It can be a linter (via plugins) to:
        (It will be seen later!)
        
        - TypeScript
        - Vue
        - React
        - Angular
        - HTML (inline <scripts>)



ESLINT > INTEGRATION WITH THE IDE: VSCODE

    1) In .vscode/settings.json file, tells VSCode which languages use ESlint:

        // CUSTOM CODING BEHAVIORS > JAVASCRIPT
        "[javascript]": {
            ...,
            "editor.codeActionsOnSave": {
                "source.fixAll.eslint": "explicit"
            }
        },
        "eslint.validate": [
            "javascript"
        ],
        "eslint.workingDirectories": [
            "frontend"  // ESLint working directories (if it's in a subfolder)
        ],

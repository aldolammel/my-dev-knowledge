

ESLINT: INSTALL AND INTEGRATION


    >> Installation:

        1) Installing ESLint core and its configuration package only as dev dependencies:

            # Using NPM:
                $ npm install -D eslint @eslint/js
            
            # Or using xxx:
                $ xxxxxxx

            # Check if everything's right on the front-end folder's package.json file!


    >> Integration:

        1) Install the ESLint extension (by Microsoft) in your IDE;

        2) In the frontend folder root, check if you already have the ESLint config file:
            /JavaScript/Linter-and-formatter-and-typechecker/eslint/examples/eslint.config.js

        3) (Optional) Add the ESLint as a script in package.json file also in the frontend folder 
                    root to make ESLint terminal commands available:
                    /JavaScript/Linter-and-formatter-and-typechecker/eslint/examples/package.json

        4) (Optional) In case of Git versioning, add it in your .gitignore file:
            ### NODE.JS ###
            ...
            # Optional eslint cache
            .eslintcache
            
        5) IDE configs:

            >> For VSCode:
                /JavaScript/Linter-and-formatter-and-typechecker/eslint/for-vscode-configs.txt

            >> For xxxxx:
                /xxxxxxxxxxxxxxxxx



PYTHON: INSTALLATION

    WINDOWS:
        Make sure to check the box that says “Add python.exe to PATH” to make Python accessible
        from the command line.


    UBUNTU:
        >> Check the python version installed:
        $ python3 --version

        # Update the local repository info:
            $ sudo apt update
        
        # Install Python, pointing the global version (it will be updated further if needed):
            $ sudo apt install -y python3.12



PYTHON > UV PACKAGE MANAGER: UNINSTALL A SPECIFIC PYTHON VERSION

    BE AWARE:
        You are able to uninstall Python through UV only if you installed Python using UV method:
            /Python/Package-manager/uv/install-python-with-uv.txt
            
        Otherwise, uninstall Python using the traditional way:
            /Python/1-python-installation/uninstall.txt
        
    
    1) Check the all Python installed through UV:
        $ uv pip tree
        # or:
        $ uv run python --version
    
    
    2) Uninstalling a specific version:
        $ uv python uninstall 3.13

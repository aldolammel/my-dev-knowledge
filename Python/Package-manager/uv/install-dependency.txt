

PYTHON > UV PACKAGE MANAGER: INSTALLING DEPENDENCIES

    PRE) What is it:
        ./_about.txt

    BE AWARE:
        Avoid to use "uv pip install <package_name>" once this command doesn't automatically update crucial files in a project like 'uv.lock' and 'pyproject.toml'.

    # Mandatory one:
        $ uv add <package_name>
    
    # Optional one:
        $ uv add --optional <sub-group> <package_name>
        # E.g. $ uv add --optional dev ruff

    # Check installed dependencies:
        $ uv pip list

                    >> Make sure you're in the project virtual environment!
                        
                        >> On Ubuntu/Windows:

                                $ pip install django

                            >> Check the current django installed:
                                $ python -m django --version

                            >> upgrade django if needed;
                                $ python -m pip install --upgrade pip

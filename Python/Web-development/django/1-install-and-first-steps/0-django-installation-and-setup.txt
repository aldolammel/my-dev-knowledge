    5) Now, in the project root folder, you must see some files, like 'manage.py', including the folder 'core' with its content. If so, you create the django project correctly! That said, keep going:
        
        5.PRE.1) Assuming you already are in the project environment!

        5.PRE.2) (if applicable) Assuming you already are in the right Versioning branch!
            /Versioning/git/command-checkout.txt
        
        5.1) Test Django files:
            $ python manage.py check
            or
            $ uv run manage.py check


        5.2) Ask Django to create its db (further we'll see how to change db systems):
            
            # If no migrations files in sub-apps folders:
                $ python manage.py makemigrations
                or
                $ uv run manage.py makemigrations

            # Build the local database:
                $ python manage.py migrate
                or
                $ uv run manage.py migrate

        5.3) Run Django to check if it's accessible by your local browsers:
            # Starting Django server:
                $ python manage.py runserver
                or
                $ uv run manage.py runserver
            # On browser:
                http://localhost:8000/

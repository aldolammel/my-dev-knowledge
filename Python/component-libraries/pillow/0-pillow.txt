

PYTHON LIB: PILLOW

    Pillow is a powerful imaging library that adds image processing capabilities to your app. Here's what it does:

        - Open, manipulate, and save various image file formats;
        - Perform image processing operations;
        - Handle image uploads in web applications.



    >> Installation:

        PRE) Assuming you're in the project environment!
        
        # Using UV:
            $ uv add pillow
        
        # Or using PIP:
            $ python3 -m pip install Pillow

    INSTALL AND UNINSTALL APPS:


        IMPORTANT:
            When you run "sudo apt install <package>", APT searches through its package index â€” a db of all available packages from repositories you've already added to your system. This index is created and updated when you run "sudo apt update".

        
        >> Installing:

            $ sudo apt install -y <package_name>                  <-- '-y' skip install question.


            >> Finding a package name installed:
                ./finding-installed-apps.txt


            >> Check apt-update repo list:
                ./checking-app-is-in-repo-list.txt


        >> Removing:

            >> .deb:
                $ sudo apt remove <package_name>

                    # Clean up APT cache:
                        $ sudo apt autoclean

                    # Remove unused old apps/dependencies:
                        $ sudo apt autoremove

            >> Snap:
                $ sudo snap remove <app_name> || true

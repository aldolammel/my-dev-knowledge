

DEBIAN/UBUNTU: INSTALL AND UNINSTALL APPS

    
    >> Installing (common):
        $ sudo apt install <package_name> -y    <-- this '-y' skip the install question "yes or no".


    >> Installing .sh files:
        1. Copy the entire file name;
        2. Open terminal
        3. type and enter: bash <file name>


    >> Finding a package name installed:
        /OS/linux/debian-ubuntu/1-apps-installations/finding-installed-apps.txt


    >> Removing:
        $ sudo apt remove <package_name>
            # Clean up APT cache:
                $ sudo apt autoclean

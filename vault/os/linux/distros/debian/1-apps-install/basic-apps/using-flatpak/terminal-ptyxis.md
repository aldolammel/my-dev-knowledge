# System tool > Ptyxis Terminal (from new Ubuntu 26)

    In case you are not using the latest Ubuntu 25+.


    PRE) Assuming you already installed Flatpak support:
        .../1-apps-install/0-preparing-the-system.md


    1) Install Ptyxis:
        $ flatpak install flathub app.devsuite.Ptyxis


    2) Customize it:

        2.1) Install a compatible font and set it as the font to be used in Terminal:
            "Fira Code"

                Using: Fire Code Regular 11

        2.2) Palette = "Birds Of Paradise" or...
                        "Breath Silverfox"
                        "Broadcast"
                        "Chameleon"
                        "Solarized Darcula"
                        "Monokai Pro"


        2.3) Other customs:
            - Line Spacing = 1,1
            - Blinking Text = Focused
            - Cursor Shape = Underline



DEBIAN/UBUNTU > BUILDING UP A SERVER: REINSTALLING


    PRE.1) Make sure you knows which OS the server demands!
    
    PRE.2) Make sure you have done all server backups needed!
    
    1) On the VPS provider online panel, select the OS to be installed and do it;
    
    2) Once the new OS copy is installed:
    
        >> Have you ever worked in this VPS through your current local machine?
        
            2A) No:
                Straight on to:
                    ./1-step.txt
                
            2B) Yes:
                # Remove the old SSH host keys from your local 'known_hosts' file:
                    $ ssh-keygen -f '/home/<your_user>/.ssh/known_hosts' -R '<server_address>'
                        E.g.
                            ssh-keygen -f '/home/aldo/.ssh/known_hosts' -R 'vps123.public.com'
                            
                # Next, go to:
                    ./1-step.txt



LISTING ATTRIBUTES FROM OTHER CLASSES:


    >> In your sub-app admin.py:

        class UserProfileCMS(admin.ModelAdmin):
            list_display = (
                'id',
                'user',
                'first_name',  # Custom method from other class
                'age',
                'sex',
                'country',
                'last_login',  # Custom method from other class
            )
            readonly_fields = ('user', 'first_name', 'last_login')

            def first_name(self, obj):
                return obj.user.first_name

            def last_login(self, obj):
                return obj.user.last_login



            >> Custom methods above are fields that are NOT listed as UserProfile attributes
                in sub-app models.py file. In this example, they are coming from the 
                Django built-in class 'User'.

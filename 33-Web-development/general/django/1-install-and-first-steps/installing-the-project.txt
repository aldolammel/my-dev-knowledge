INSTALLING DJANGO:


    1) Create the project folder, and get it:
        $ mkdir <project_folder>
        $ cd <project_folder>


    2) Create the virtual environment and active it:
        <Check the folder '03-Virtual-Environment'>


    3) Update the pip:
        <Check the folder '04-Basic-Modules'>


    4) Installation:
        $ pip install django

        Only for VSCode:
        # DjLint (html template formatter for django/jinja)
        $ pip install -U djlint
        # Install the DjLint extension for VSCode.
        # Install the Django (by Baptiste) extension for VSCode.
        # In VSCode, In Html's, change the language mode to 'Django-Html'


    5) Create the Django Project Structure:
        # make sure you're in project root folder and:
        $ django-admin startproject proj_<project_name> .


    6) If you won't use the SQLite as database, do it:
        /33-Web-development/general/django/3-database-models/0-installing-and-adminUser/1-define-your-database/


    7) Set the environment variables and create your .env file:
        /33-Web-development/general/django/1-install-and-first-steps/environment-variables-django.py


    8) Make sure you're in the same folder the 'manage.py' file, and then start the django for testing purposes:
        $ python manage.py runserver


    9) Create the 'templates' folder in the project-folder (main-app folder).
        Don't mistake with the config-folder, e.g. /project-folder(main-app)/config-folder(same-name-of-the-project)/;

        E.g. Django Project Structure:

            Project-folder (main-app or root folder)
                config-folder (same name of the project)
                    settings.py (where you register each sub-app in use)
                    ...
                sub-app1-folder
                    ...
                sub-app2-folder
                    ...
                sub-app3-folder
                    ...
                ...
                manage.py


    10) Create your first sub-app in the main-app:

        10.1) Show to Django where will be each template folder in your sub-apps, going to the config-folder 'settings.py' file
            and editing this line in 'template' list:
                From:
                    'DIRS': [],
                To:
                    'DIRS': [os.path.join(BASE_DIR, 'templates')],

        10.2) Do this: /33-Web-development/general/2-creating-and-deleting-apps/creating.txt


    11) Creating the admin user:
        $ python manage.py createsuperuser
        # Now, try to login, making sure the main-app is running:
        $ python manage.py runserver
        http://127.0.0.1:8000/admin/


    12) Optionals:

        12.1) Will you use file upload?
            /33-Web-development/general/django/5-uploads/files/

        12.2) Will you use imagem upload?
            /33-Web-development/general/django/5-uploads/images/

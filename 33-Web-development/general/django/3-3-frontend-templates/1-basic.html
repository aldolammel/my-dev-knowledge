E.g. {# Calling another template as base for this template #} {% extends 'base.html' %} {# Defining the content block of the template must be loaded here #} {% block content %} {% endblock content %} {# A conditional example #} {% if cars %}

Good, there is at least one car available!

{% else %}

Oops, there is no car available!

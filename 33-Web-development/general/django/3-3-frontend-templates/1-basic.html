DJANGO TEMPLATES: BASIC >> Percentage signs within curly-braces {% %} are TEMPLATE-TAGS used to perform logic operations, control structures, and include other template functionalities within a template. E.g. {% extends 'base.html' %} {% block content %} {% endblock content %} {% if cars %}

Good, there is at least one car available!

{% else %}

Oops, there is no car available!

{% endif %} {% for car in cars %}

xxxxxxxx

{% endfor %} >> Double curly-braces {{ }} are TEMPLATE-VARIABLES which are used to output the value of a variable sent by the CONTEXT (dictionary) from a views.py file. E.g. {{ car }} OR {{ car.name }} OR {{ car['name'] }} {{ car.brand }} OR {{ car['brand'] }} {{ car.year }} OR {{ car['year'] }} >> A perfect example of using the template you find in a project example that I create to examplify the usage of views: /33-Web-development/general/django/3-2-backend-views/project-example/sub-app/templates/

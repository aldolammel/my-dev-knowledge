
POSTGRESQL

    1) Install the PostgreSQL Python module:
        https://www.psycopg.org/


    2) If the database won't be in the cloud, install the PostgreSQL locally:
        Go to the folder: 34-database/PostgreSQL/0-Installing/

    
    3) Go to your project folder and open the settings.py file and add these lines
        in the right place:

            import os
            from dotenv import load_dotenv

            # Quick-start

            load_dotenv()  # take environment variables from .env. To force this to override the environment variables, use load_dotenv(override=True)

            HOST = os.getenv('HOST')
            PORT = os.getenv('PORT')
            DB_USER = os.getenv('DB_USER')
            DB_PWD = os.getenv('DB_PWD')
            DB_NAME = os.getenv('DB_NAME')
            DB_HOST = os.getenv('DB_HOST')
            DB_PORT = os.getenv('DB_PORT')

            # SECURITY WARNING: keep the secret key used in production secret!
            SECRET_KEY = os.getenv('SECRET_KEY')


            # Database

            DATABASES = {
                "default": {
                    "ENGINE": "django.db.backends.postgresql",
                    "NAME": DB_NAME,
                    "USER": DB_USER,
                    "PASSWORD": DB_PWD,
                    "HOST": DB_HOST,
                    "PORT": DB_PORT,
                }
            }

    
    4) Make a copy of '.env' file to the project root folder:
        
            /31-Environment-Variables/

        >> customize the .env as needed following the step-3 variables right above!


    5) If you see the file sqlite.db in the project root folder, delete it;

    
    6) Give the order to build the database:
         $ python manage.py migrate

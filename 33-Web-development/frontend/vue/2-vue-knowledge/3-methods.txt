    >> To understand 'Options API' approach:
        /33-Web-development/frontend/vue/2-vue-knowledge/3-component-approach.txt
    
        In Vue.js, methods are functions defined inside a Vue instance's methods option. They are
        used to perform specific tasks, such as responding to user interactions (e.g., clicks),
        processing data, or executing any custom JavaScript logic. Methods can be called directly in
        your Vue templates or elsewhere in your component.

        # <script>:
        const MyProjectThing = {
            data() {
                return {
                    ...
                }
            },
            methods: {
                ...HERE YOUR CODE!
            }
        };

        >> Example 1:
            # <script>:
            const MyProjectThing = {
                data() {
                    return {
                        ...
                    }
                },
                methods: {
                    total: function() {
                        // Declarations:
                        var total = 0;
                        // For looping:
                        this.products.forEach(function(product){
                            // Only those products actived:
                            if (product.isActive) {
                                total += product.price * product.quantity;
                            }
                        });
                        // Returning result rounded yet:
                        return total.toFixed(2);
                    }
                }
            };

            # <template>:
            {{ total() }}

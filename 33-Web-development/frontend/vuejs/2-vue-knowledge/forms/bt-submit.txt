

VUE WITH FORMS: SUBMIT BUTTON

    To make a real clickable button, you need to use the directive 'v-on' (@):
        /33-Web-development/frontend/vuejs/2-vue-knowledge/v-on.txt

    E.g.

        # On <script> layer:

            export default {                
                data() {
                    return {
                        chosenAnswer: undefined,
                    }
                },
            
                methods: {
                    submitForm() {
                        if (!this.chosenAnswer) {
                            alert("Please, select an answer!");
                        }

                    }
                }
            }
        

        # On <template> layer:
            
            <button
                type="button"
                @click="submitForm"
            >
                Submit
            </button>

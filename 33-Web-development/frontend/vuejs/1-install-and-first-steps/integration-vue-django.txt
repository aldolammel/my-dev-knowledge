

    INTEGRATING VUE WITH DJANGO:

        >> Reminder: basic steps about Vue.js:
            /33-Web-development/general/vuejs/1-install-and-first-steps/installation.txt

        >> If you are ready, let's go:


            1) Install Vue-Command-Line-Interface (Vue CLI):
    
                >> Win:
                    $

                >> Ubuntu:
                    $ sudo npm install -g @vue/cli

                >> Mac:
                    $

                1.1) (Optional) Make sure you got the .gitignore updated:

                    ### Vuejs ###
                    dist/
                    npm-debug.log
                    yarn-error.log


            2) Create a new Vue project folder inside the Django project:
            
                >> Win:
                    $

                >> Ubuntu:
                    $ vue create <frontend_folder_name>
                    # e.g. vue create frontend

                >> Mac:
                    $

            3) Build up the Vue project:
            
                >> Win:
                    $
                    
                >> Ubuntu:
                    $ cd <frontend_folder_name>
                    $ sudo npm run build
                
                >> Mac:
                    $

            4) Integrate the Vue and Django! Move the contents of the 'dist' directory to a
                new static directory within your Django project:

                >> Win:
                    $
                    
                >> Ubuntu:
                    # Already in <frontend_folder_name>:
                    $ sudo mv dist ../static
                
                >> Mac:
                    $
                    
            5) Update the settings.py:

                STATIC_URL = 'static/'
                STATICFILES_DIRS = [
                    BASE_DIR / "static",
                ]


            X) Update your main template header:

                <!-- CSS -->
                {% load static %}
                <link href="{% static 'css/<vue_css_name_here>.css' %}" rel="stylesheet" />



            X) Run both servers in different terminals with the right virtual env activated:

                # In django folder:
                $ python manage.py runserver

                # In vue folder:
                $ sudo npm run serve

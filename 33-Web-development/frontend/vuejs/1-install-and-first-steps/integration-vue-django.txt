        >> Reminder: basic steps about Vue.js:
            /33-Web-development/general/vuejs/1-install-and-first-steps/_installation.txt


        >> Installing Vue:

            X) Go to the Django Project folder and execute this command:
                $ npm create vue@latest

                # Config:
                Project name = vue-<client_name>
                ESLint = Yes!

                # Check the Vue version installed:
                $ npm list vue

            X) Install the Vue project dependencies and start the dev server:
                $ cd vue-<client_name>
                $ npm install
                $ npm run dev

                # Run the localhost to check the default page running.

            X) (optional) Official Vue extension for VSCode:
                https://marketplace.visualstudio.com/items?itemName=Vue.volar


            X) (Your project on Prod) When you are ready to ship your app to production,
                run the following:

                $ npm run build

                # This will create a production-ready build of your app in the project's ./dist directory.




        >> Integrating Vue and Django:


            X) Integrate the Vue and Django! Move the contents of the 'dist' directory to a
                new static directory within your Django project:

                >> Win:
                    $
                    
                >> Ubuntu:
                    # Already in <frontend_folder_name>:
                    $ sudo mv dist ../static
                
                >> Mac:
                    $
                    
            X) Update the settings.py:

                STATIC_URL = 'static/'
                STATICFILES_DIRS = [
                    BASE_DIR / "static",
                ]


            X) Update your main template header:

                <!-- CSS -->
                {% load static %}
                <link href="{% static 'css/<vue_css_name_here>.css' %}" rel="stylesheet" />



            X) Run both servers in different terminals with the right virtual env activated:

                # In django folder:
                $ python manage.py runserver

                # In vue folder:
                $ sudo npm run serve

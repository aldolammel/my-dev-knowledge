

RUNNING A VUE APP:


    1) On terminal, go to the project's folder!


    2) Check if there's Git version control:
        $ git status

        2.1) If so, be aware if you are in the right branch to use the application!
    

    3) Run the app:

        >> Using the front-built-tool VITE:
            $ npm run dev

            >> Check the app through the browser:
                http://localhost:5173/

        >> Using the front-built-tool Vue-CLI:
            $ npm run serve
            or
            $ yarn serve

            >> Check the app through the browser:
                http://localhost:8080/

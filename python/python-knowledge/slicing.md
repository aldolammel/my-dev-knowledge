# Slicing

---

Slicing is a clean way to extract a specific subset (a "slice") of elements from a sequence, like a string, list, or tuple, without changing the original data. Instead of writing a loop to grab items, you use a simple bracket notation with up to 3 arguments separated by colons: `sequence[start:stop:step]`.

Slicing works on any *sequence type* in Python, not just *strings*.

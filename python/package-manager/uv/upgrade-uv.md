# Updating UV itself

It's important the UV be installed globally in your local machine!


---

## Before:
1. Assuming you are NOT activated in a project venv!
2. Check if your UV is installed globally, using *uv --version*. If the UV version is shown, you're good!


---

## 1) Upgrading:
It should update the UV for use in all local projects!
```
uv self update
```

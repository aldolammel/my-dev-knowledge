# Node.js

*Node.js* is a runtime and open-source *JavaScript* environment that allows you to run JS code on the server side. This enables JS to be used for building fast, scalable server-side apps. That said, Node.js is a cross-platform that can be ran through all wide-known OS's, executing JS code outside a browser. Technically, *Node.js* is NOT a server itself! It just execute JavaScript code outside of a web browser. While it isn't a server by definition, it is most commonly used to build servers.

It's predominantly used for back-end development but also plays a significant role in front-end development, including non-web-development projects like Game, [LoT](dev-concepts/lot.md), and other types of software.

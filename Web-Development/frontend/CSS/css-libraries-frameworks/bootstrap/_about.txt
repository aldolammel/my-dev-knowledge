

WHAT IS BOOTSTRAP:

    Build fast, responsive sites with Bootstrap Powerful, extensible, and feature-packed frontend
    toolkit (html+css+JavaScript). Build and customize with Sass, utilize prebuilt grid system and
    components, and bring projects to life with powerful JavaScript plugins.

    Bootstrap is a component-based framework that provides pre-built components like buttons, cards,
    navbars, and modals. It uses semantic class names!

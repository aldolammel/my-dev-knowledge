    >> CSS Pre-processors using Vue + Vite:

        >> .scss/.sass (a Vite built-in solution):
            
            1) Associating the Vite built-in solution to the project (only for dev environment):
                $ npm add -D sass-embedded
                    # Verify your SASS dependency in package.json in the root project-folder:
                        E.g.
                            {
                                "devDependencies": {
                                    "sass-embedded": "^1.89.1",
                                }
                            }

            2) In front-end folder, open the vite.config.js file:

                export default defineConfig({
                    //...
                    build: {
                        //...
                    },
                    plugins: [
                        //...
                    ],
                    resolve: {
                        //...
                    },
                    publicDir: "public",
                    css: {
                        preprocessorOptions: {
                            scss: {
                                //...
                                quietDeps: true, // Handling deprecation warnings!
                                sassOptions: {
                                    quietDeps: true, // Belt and suspenders approach!
                                },
                            },
                        },
                    },
                    server: {
                        //...
                    },
                });
        
            3) In all Vue components (.vue) using CSS, include the language in Style tag:
                
                E.g.:
                    # .scss/.sass
                    <style lang="scss" scoped>


            >> More about Vite:
                https://vite.dev/

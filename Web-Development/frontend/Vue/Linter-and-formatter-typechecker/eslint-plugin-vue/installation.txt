

VUE > TEMPLATE LINTER > ESLINT: INSTALLATION


    PRE.1) Assuming you have done the first steps to create a Vue project:
        /Web-Development/frontend/Vue/1-install-and-first-steps/create-vue-project.txt

    PRE.2) Assuming you already are in the frontend folder of the project on terminal;
    
    PRE.3) Assuming you already installed the ESLint for JavaScript/TypeScript:
        /JavaScript/Linter-and-formatter-and-typechecker/eslint.txt


    - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 


    1) Install the ESLint plugin for Vue (eslint-plugin-vue) as development dependency:
        
        # Using NPM:
            $ npm install -D eslint-plugin-vue
    
        # Or using XXX:
            xxxxxxxx

        # Check if everything's right on the front-end folder's package.json file!

    2) Edit the ESLint configuration file on the Vue project root ('frontend', e.g.):
        /Web-Development/frontend/Vue/Linter-and-formatter-typechecker/eslint-plugin-vue/examples/eslint.config.js

    3) In vite.config.js file:
        <...>
        import eslintPlugin from 'vite-plugin-eslint'
        <...>
            plugins: [
                vue(),
                <...>,
                eslintPlugin({
                    include: ['src/**/*.js', 'src/**/*.vue', 'src/*.js', 'src/*.vue'],
                    exclude: ['node_modules/**', 'dist/**'],
                    cache: false
                })
            ],



AXIOS: INTEGRATION WITH VUE.JS:

    >> Axios installation:
        /JavaScript/NodeJS/component-libraries/axios-api/axios.txt

    >> Integration with Vue:

        >> For SPA projects:

            1) In Django .env file, and, then, in 'FRONTEND SETTINGS', add this line below:
                VITE_BASE_URL=http://localhost:<django's port>  # Under Production, change it to the client PROD_BASE_URL!

                >> CRITICAL: Vue with Vite mandatorily require env vars with 'VITE_' prefix!

                >> Need to create an .env file?
                    /Environment-Variables/

            2) In the front-end main-app-js file, include:
                // After the basic importations:
                import axios from 'axios';

                // App config, after the App creation:
                axios.defaults.baseURL = import.meta.env.VITE_BASE_URL;  // Where Django's running currently!
